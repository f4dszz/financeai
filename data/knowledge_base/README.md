# Knowledge Base Directory

## Purpose / 用途

This directory stores regulatory documents for RAG (Retrieval-Augmented Generation).

本目录存放用于RAG检索增强生成的监管文档。

## Accepted Formats / 支持格式

- PDF (.pdf)
- Word (.docx)
- Text (.txt)

## Week 2 Target / 第2周目标

- Minimum 5 PDF documents (hard acceptance criterion)
- 至少5份PDF文档（硬验收标准）

## Example Documents / 示例文档类型

- Banking regulations / 银行业监管法规
- Securities compliance guides / 证券合规指南
- Risk management policies / 风险管理政策
