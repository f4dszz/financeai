# Sample Charts Directory

## Purpose / 用途

This directory stores financial chart images for OCR extraction.

本目录存放用于OCR提取的金融图表图片。

## Accepted Formats / 支持格式

- PNG (.png)
- JPEG (.jpg, .jpeg)
- WebP (.webp)

## Week 3 Target / 第3周目标

- Minimum 10 chart images (hard acceptance criterion)
- 至少10张图表图片（硬验收标准）

## Supported Chart Types / 支持图表类型

- K-line / candlestick charts (K线图)
- Bar charts (柱状图)
- Line charts (折线图)

## Naming Convention / 命名规范

```
{type}_{source}_{date}.{ext}
Example: kline_stock_20241201.png
```
